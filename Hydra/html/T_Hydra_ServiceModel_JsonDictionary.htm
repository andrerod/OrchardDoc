Remarks
If PrimitiveKeyName and PrimativeValueName are set, describes an array of objects.
Copy
{ 'dict': [
  { 'keyName': 'element1', 'valueName': 'value1'},
  { 'keyName': 'element2', 'valueName': 'value2'}
}
Otherwise, a collection of objects.
Copy
{ 'dict': {
  'element1': 'value1',
  'element2': 'value2'
  }
}

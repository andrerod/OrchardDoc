ForEachSeparatedT(IEnumerableT, ActionT, Action)
Maps an action with side effects over a sequence but runs a separating action in between each element. For example, if a mapping action prints the elements of a sequence then the separting action might be used to print commas.
ForEachSeparatedT(IEnumerableT, ActionT, Action, Action)
Maps an action with side effects over a sequence but runs a separating action in between each element with a distinct separator for the last two elements. For example, if a mapping action prints the elements of a sequence then the separting action might be used to print commas with a last separator of a ", and".

ContextualInvalidOperationException
Processing large object graphs can often lead to confusing and/or unhelpful exception messages because an ArgumentNullException deep in a recursive call without parameter values can be meaningless. ContextualInvalidOperationException is used by the ErrorContext class which allows you to wrap any exception with helpful context to make the error more discoverable. If multiple ErrorContext calls are triggered as an exception bubbles outward, they'll keep appending their context to the same ContextualInvalidOperationException to prevent a hierarchy of useless InnerExceptions.

Reflected Class
The Reflected modeling attribute indicates that a model property should take its value from a property of its reflected instance. This is commonly used to set something like a Service.Name property to the corresponding Type.Name property of the reflected Type instance used to create the Service model. Note that the reflected value will not be used if the property already has a value.

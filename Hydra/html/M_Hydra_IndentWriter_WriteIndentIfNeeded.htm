IndentWriterWriteIndentIfNeeded Method
Writes an indent if needed. This is used before each write operation to ensure we're always indenting. We don't need to indent for a series of calls like Write("Foo"); Write("Bar"); but would indent between a series like WriteLine("Foo"); Write("Bar");
